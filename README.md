## üìå How To Install?

- Clone This Repo
```
git clone https://github.com/eexxprd/black-dragon-coffeeshop-frontend
```
- Go To Folder Repo
```
cd black-dragon-coffeeshop-frontend
```
- Install Module
```
npm install
```
- Type ``` npm run start ``` To Start The Project
